IMPORTANTE: Agrega aquí las imágenes de banderas o screenshots

Puedes agregar:
- Banderas en formato PNG (si no usas emojis)
- Screenshots del plugin funcionando
- Icono del plugin (icon.png - 256x256px)
- Banner del plugin (banner.png - 1544x500px)

Las banderas con emojis Unicode ya están incluidas en el código,
así que no es necesario agregar imágenes de banderas a menos que
quieras personalizarlas.

Para usar imágenes de banderas en lugar de emojis:
1. Descarga banderas de: https://flagicons.lipis.dev/ o https://www.flaticon.com/
2. Guárdalas aquí con nombres: USD.png, EUR.png, CAD.png, etc.
3. Modifica el archivo includes/class-api-client.php para usar las rutas de imágenes
